# data-collector

## Description

This module is dedicated to fetch data from `data-generator` module and analyze or store it.

## Technologies

- [Golang](https://golang.org/) v1.14

## How to use

### Installation

```bash
make build
```

### Running

```bash
make run
```

### Testing

```bash
make test
```

### Cleaning up

```bash
make clean
```
